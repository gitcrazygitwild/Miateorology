Method (MVP)

  • Live: scores = average absolute error of daily high/low (°F), against recent NWS station observations.
  • Backfill: Open-Meteo only, comparing archived forecasts vs historical weather (reanalysis).
  • Blend: simple average of available sources at each date/time.
  • Disagreement: spread between provider forecasts for each upcoming day.
  • Conditions: each source is normalized into simple buckets like rain, snow, thunder, precip chance, and summary.
